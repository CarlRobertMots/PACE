## About The Project

**PACE** is a React Native mobile application built with Expo that gamifies the fitness experience. We believe that exercise shouldn't be a chore—it should be a quest.

This app combines a real-time pedometer with RPG mechanics. Every step you take deals damage to a "Boss". The more you walk, the faster you defeat the enemy and level up.

Key features include:

- **Gamified Activity:** Visual Boss Health bars that deplete as you walk.
- **Retro Aesthetic:** Immersive dark mode UI with pixel-art fonts, glassmorphism, and neon shadows.
- **Detailed Analytics:** Track your Day, Week, and Month with interactive circular charts and "Ray" graphs.
- **Cross-Platform:** Optimized for both iOS and Android (including custom Safe Area handling).
